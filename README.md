# LinkedIn Automation Workflow (Free System)

A complete system to automate your LinkedIn growth, content creation, and lead management using free tools.

---

## 🚀 What This System Does

- Generate LinkedIn content using AI
- Store and manage content in Google Sheets
- Schedule posts manually or semi-automatically
- Capture leads from comments and messages
- Track and follow up leads automatically

---

## 🧩 Tools Used

- Make (Integromat)
- Google Sheets
- ChatGPT / AI tools
- LinkedIn

---

## 🔄 Workflow Overview

1. Content ideas generated using AI  
2. Stored in Google Sheets  
3. Converted into ready-to-post format  
4. Posted manually or via scheduler  
5. Leads captured from engagement  
6. Stored in CRM (Google Sheets)  
7. Follow-up automation triggered  

---

## 📊 System Architecture

Content → Sheet → Post → Engagement → Lead Capture → CRM → Follow-up
